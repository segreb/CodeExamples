Это не проект, просто отдельные модули. 
В DentalDataFeeders.pas и DentalObjectModels.pas содержится "фреймворк", 
который я писал для себя чтобы упростить
себе типовые операции CRUD с записями базы данных. 
В GoogleCalendarAPI.pas - мой движок по заполнению календаря из аккаунта Google, 
включая авторицию по OAuth.
uUserScheduleThread.pas - это рабочая нитка сервиса, 
который синхронизирует расписание пользователя,
хранящееся в моей программе и календарь из aккаунта пользователя в Google.
PaymentsOM.pas - модуль для учёта платежей по ранее выписанным счетам.
